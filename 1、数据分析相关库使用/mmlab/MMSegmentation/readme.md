# `MMSemgentation`


## `MMSemgentation 安装：`

* `1、`准备好 `MMCV` 环境

* `2、`下载 `MMSegmentation` 项目：https://github.com/open-mmlab/mmsegmentation
  
* `3、`在虚拟环境中安装 `mmseg` 包：

        如果 conda install mmseg 或者 pip install mmseg 失败
        可以使用：
        pip install -i https://pypi.tuna.tsinghua.edu.cn/simple mmsegmentation
