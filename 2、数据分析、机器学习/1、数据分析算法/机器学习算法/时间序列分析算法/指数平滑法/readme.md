# `指数平滑法`


在做时序预测时，一个显然的思路是：认为离着预测点越近的点，作用越大。比如我这个月体重 `100` 斤，去年某个月 `120` 斤，显然对于预测下个月体重而言，这个月的数据影响力更大些。假设随着时间变化权重以指数方式下降——最近为 `0.8` ，然后 `0.8**2` ，`0.8**3…` ，最终年代久远的数据权重将接近于 `0`。

`将权重按照指数级进行衰减，这就是指数平滑法的基本思想。`
